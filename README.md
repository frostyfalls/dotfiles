# dotfiles

These are my personal configuration files; things may change without notice. Shoddy organization ahead, you have been warned.

## Dependencies

- [XStow](https://xstow.sourceforge.net/)
- [Make](https://www.gnu.org/software/make/)

## Usage

Installation:

```
make install
```

Uninstallation:

```
make uninstall
```
